1. input type="time" 시간(오전/오후 시 분)을 지정하는 폼요소


2. input type="date" 날짜(년월일)를 지정하는 폼요소


3. input type="datetime-local" 날짜와 시간(년월일 오전/오후 시 분)을 지정하는 폼요소


4. input type=" week" 날짜(년 주)를 지정하는 폼요소


5. input type="month" 날짜(년 월)를 지정하는 폼요소


6. input type="number" 숫자값을 지정하는 폼요소

min/max로 입력값의 최소/최대를, step로 증감 간격을

명입니다 (min="0")
명입니다 (max="10")
명입니다 ()

7. input type="range" 범위내 값을 지정하는 폼요소


8. input type="tel" 연락처(전화번호)를 지정하는 폼요소

모바일 브라우저에서 가상키패드가 숫자형식으로 바뀔 수 있고 전화걸기 가능

9. input type="email" 연락처(이메일)를 지정하는 폼요소

브라우저에서 알맞은 이메일형식 인지 체크 해줌(@유무,@이후 작성 여부)

10. input type="url" 연락처(웹주소)를 지정하는 폼요소

브라우저에서 알맞은 웹주소형식 인지 체크해줌(프로토콜 http://, https:// 유무)
